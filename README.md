# YOLO + SAHI Inference

Проект для летней практики. Используется библиотека Ultralytics для инференса YOLOv8 и SAHI для детекции мелких объектов на изображениях высокого разрешения через веб-интерфейс.

## Установка (WSL / Linux)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Посе того как все установиться выполняем 
```bash
python app.py
```
После того как программа запуститься будет предоставлен адрес локального сервера, который нужно открыть в браузере

Можно будет выбрать модель, которую вы хотите использовать, или указать путь к своей

Также доступны: выбор устройства обработки, настройка SAHI

После загрузки фото, выбора модели и нажатия кнопки "запустить детекцию", если выбранной модели нет на устройстве
она скачается с GitHub и после инициализации выполнится.
